| Format | Hex Value   | Explanation                          |
|--------|-------------|--------------------------------------|
| RGBA   | 0xFF0000FF  | Red first, Alpha last                |
| ARGB   | 0xFFFF0000  | Alpha first, then Red, Green, Blue   |
| BGRA   | 0x0000FFFF  | Blue first, Red next to least byte   |
| ABGR   | 0xFF0000FF  | Alpha first, Red in least byte       |
